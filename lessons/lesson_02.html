Handling Large Datasets and Data Wrangling

Overview

In this lesson, we'll explore different structured data formats and their usages and how we can create, collect and inspect data in a quantitative and scalable way.

Materials

  • Structured Data Formats - slides
  • Cheat sheet for the command line - notes. Alternatively the notes are also published at Per's Notes - Command Line Cheat Sheet for Data Wrangling. The latter version might get updates.
